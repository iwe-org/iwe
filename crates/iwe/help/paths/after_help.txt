OUTPUT FORMAT:

  Each line shows a path with segments separated by " • ":

    Root Document
    Root Document • Child Section
    Root Document • Child Section • Nested Item
    Another Root

  Path segments are document or section titles. Paths are sorted
  alphabetically and deduplicated.

EXAMPLES:

  # Show paths up to 4 levels deep (default)
  iwe paths

  # Show only top-level documents
  iwe paths --depth 1

  # Show deeper hierarchy
  iwe paths --depth 6

  # Count paths at each depth
  iwe paths | awk -F ' • ' '{print NF}' | sort | uniq -c
